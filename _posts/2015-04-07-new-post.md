> 将ISO转为dmg

```
$ hdiutil convert -format UDRW -o linux.dmg linux.iso
```

> 插入U盘 查看U盘的设备名

```
diskutil list
```

> 卸载U盘，不是退出

```
diskutil umountDisk /dev/disk
```

> 写入镜像

```
sudo dd if=linux.dmg of=/dev/rdisk1 bs=1m
```

> 退出

```
diskutil eject /dev/disk1
```
